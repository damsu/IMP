SOME STUFF

RWD:

-> Responsive Web Design aims at making each webpage feel like it was design for the support used to see it through. It is used to properly scale web elements such as texts, images, videos, etc... according to the screen or window size. Before RWD, webpages were designed for typical computer screen size and everything had a fixed scale.

-> CSS helps building responsive websites by allowing the designer to decide the sizes and styles of every element on page, and particularly to use scaling not in pixels but in percentage of width, height, etc... thus making elements responsive to window size.

-> em based sizing sizes according to text size, meaning that sizing a margin with em will make the margin somewhat proportionnal to text size, making the text more readable.
