1. About jQuery

  • jQuery allows the developper to simplify, speed-up and extend the use of javascript. You can select and manipulate DOM elements easier, handle animations as well as Ajax requests.

  • It helps the developper to achieve better results in a quicker time (usually reduce by a lot the amount of code needed to do something).

  • The main issue with Persons List that jquery would help solve is selecting and appending content within the DOM element of the list.

  • CDNs are Content Delivery Networks. They cover a large spectrum of content delivery services but in the case of jQuery, a CDN will provide a link the libraries that the developper can just add, instead of downloading the libray.

  • It helps mainly because it saves the hassle of downloading and saving the library in the project, also because it's fast and reliable, and it is fairy easy to just add/remove links, to try different versions, different frameworks, libraries, etc...
